Graffiti Trials v2.0: Patch Notes

November 12, 2025

Graffiti Trials just got its biggest update yet, bringing a more polished and fun gameplay experience.

This update focuses on improving interactions with new sounds and animations, and enhancing the overall user experience through UI updates and character improvements.

Updates

  • Added sounds for several interactions
  • Added new animations for interacting with non-playable characters
  • Added a new tutorial and guides, delivered by a new in-game friend
  • Added a graffiti can icon that shows how much spray paint you have
    • left
  • Added a completion indicator for time trials
  • Added a title screen to start the game with options to play, view stats, or change options
  • Added a pause menu
  • Added cursor locking while playing and unlocking while interacting with UI
  • Added trees near Course 1
  • Modified Course 1's level design
  • Modified font style
  • Enhanced controller support and input device switching
  • Enhanced quality of shadows

Bug Fixes

  • Fixed an issue where jumping near the time trial master and interacting with them immediately could crash the game
  • Fixed unintended higher jumps near non-playable characters
  • Fixed a typo in the time trial master dialogue

Thank you to everyone who playtested the game and shared feedback. Everyone had great and inspiring ideas for the future of Graffiti Trials. If you have any more, feel free to message me! I had a lot of fun developing this update and learning along the way.

The time trial master is waiting for a worthy opponent. Go practice and beat 'em! You never know when more of them may appear...
